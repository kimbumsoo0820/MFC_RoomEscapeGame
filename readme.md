## DREAM MAKER 방탈출 게임 

>게임 이름 : Dream Maker
>스토리 :  오랜시간동안 의식을 잃은 상태의 주인공이 악몽으로부터 벗어나고, 잃어버렸던 기억을 되찾으며 자신의 트라우마를 직면하여 치유하고, 의식을 되찾게 되는 스토리 이다.
>설명 : 단서를 얻고, 단서를 통해 문제를 풀어나가며 진행 하는 게임
>기술: C++
>기여도 : 80%
